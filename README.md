# Brain-to-text '25 (The GRAD-Descenters)

This repository contains the codebase and model checkpoints for our entry to the [Brain-to-text '25 challenge](https://www.kaggle.com/competitions/brain-to-text-25) on Kaggle. For more information about the data and baseline setup, see the [competition hosts' GitHub repository](https://github.com/Neuroprosthetics-Lab/nejm-brain-to-text).
